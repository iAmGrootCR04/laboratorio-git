Introducción a Git

Guía rápida para el control de versiones

¿Qué es Git?

Git es un sistema de control de versiones distribuido, diseñado para manejar todo, desde proyectos pequeños hasta muy grandes, con rapidez y eficiencia.

Comandos Esenciales

Comando Descripción
git init Inicializa un nuevo repositorio local.
git add . Prepara todos los cambios para el próximo commit.
git commit -m "mensaje" Registra los cambios en el historial.
git status Muestra el estado actual de los archivos.
git push Sube los cambios al repositorio remoto.

El Ciclo de Trabajo

El flujo básico consiste en modificar archivos, añadirlos al Staging Area y luego confirmar los cambios con un commit.
